🎨 آموزش خصوصیات Background در CSS

Background-color | Background-image | Background-repeat

🔍 نمونه‌های عملی

background-color: #e74c3c
background-color: rgba(231, 76, 60, 0.5)
background-image: url('image.jpg')
background-image: pattern
background-repeat: repeat
background-repeat: repeat-x
background-repeat: repeat-y
background-repeat: no-repeat

📝 خلاصه و نکات کلیدی

🎯 Background-color:

.element {
  background-color: #3498db;
  background-color: rgba(52, 152, 219, 0.5);
  background-color: transparent;
}

🖼️ Background-image:

.element {
  background-image: url('path/to/image.jpg');
  background-size: cover;
  background-position: center;
}

🔁 Background-repeat:

.element {
  background-repeat: repeat; /* تکرار کامل */
  background-repeat: repeat-x; /* تکرار افقی */
  background-repeat: repeat-y; /* تکرار عمودی */
  background-repeat: no-repeat; /* بدون تکرار */
}

💡 نکات مهم برای دانشجویان

  • ✅ Background-color همیشه در زیر Background-image نمایش داده می‌شود
  • ✅ می‌توانید از چندین عکس به عنوان پس‌زمینه استفاده کنید
  • ✅ مقدار پیش‌فرض background-repeat برابر با repeat است
  • ✅ برای عکس‌های بزرگ از no-repeat استفاده کنید
  • ✅ برای پترن‌های کوچک از repeat استفاده کنید
  • ✅ از background-size: cover برای پوشش کامل زمینه استفاده کنید
