آموزش ویژگی Resize در CSS

انواع resize روی divها:

resize: none
تغییر اندازه این باکس غیرفعال است
resize: both
می‌توانید این باکس را در هر دو جهت (عرض و ارتفاع) تغییر اندازه دهید
resize: horizontal
فقط می‌توانید عرض این باکس را تغییر دهید
resize: vertical
فقط می‌توانید ارتفاع این باکس را تغییر دهید
